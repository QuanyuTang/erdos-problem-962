# An Improved Lower Bound on Erdős Problem #962

This repository contains the note:

> *An Improved Lower Bound on Erdős Problem #962*

## Abstract
We prove a new lower bound for Erdős Problem #962, showing that $k(n) \ge \exp\left(\left(\frac{1}{\sqrt{2}} - o(1)\right)\sqrt{\log n \log \log n}\right).$
